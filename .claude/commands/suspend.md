作業を安全に中断するための「セーブ」を行います。

## 手順

1. **WIPコミット**: 未保存の変更がある場合
   ```bash
   git add .
   git commit -m "wip: suspending work [skip ci]"
   ```

2. **プッシュ**: `git push` でリモートに同期

3. **コンテキスト保存**: Linearの現在のタスクのコメントに以下を投稿（MCP使用）：

   ```
   ## 作業中断メモ
   - **ブランチ**: [現在のブランチ名]
   - **ここまでやったこと**:
     - [箇条書き]
   - **Next Action**: [次にやるべきこと]
   - **残っている懸念点**: [あれば記載]
   ```

4. **完了報告**: 「お疲れ様でした。Linearに状況を記録しました。次回は `/resume` で再開できます。」

## 意図
AIのメモリがリセットされても、Linearを見れば「何をすべきか」が残っている状態を作る。
